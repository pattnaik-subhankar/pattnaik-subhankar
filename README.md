# Subhankar Pattnaik

### Senior Azure PaaS Engineer → Solution Architect | 6+ Years | LTIMindtree

> **I build and integrate Azure PaaS services at enterprise scale — shaped by 2.5 years of diagnosing what breaks in production.**

Before I engineered solutions, I spent 2.5 years in Microsoft Azure Support resolving Sev-A and Sev-B incidents for global enterprise clients. Production AKS clusters with memory leaks. DNS failures across Private Endpoints. Firewall rules that took down payment gateways. That frontline experience is what separates me from an architect who's only ever read the docs — I know what fails, and I build accordingly.
